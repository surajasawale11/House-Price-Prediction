# Boston Housing Price Prediction

## Overview

This project predicts house prices using the Boston Housing Dataset and a Linear Regression model. The objective is to estimate house prices based on various housing and neighborhood characteristics.

## Dataset

The Boston Housing Dataset contains 506 records and 13 features. The target variable is **MEDV**, which represents the median value of owner-occupied homes.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Methodology

The dataset is loaded and explored to understand its structure and distributions. Missing values are handled during preprocessing, and feature scaling is applied using StandardScaler.

The dataset is split into training and testing sets. A Linear Regression model is trained on the training data and then used to predict house prices on unseen test data.

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

## Visualizations

* House Price Distribution
* Actual vs Predicted Prices
* Residual Plot
* Feature Importance Analysis

## Skills Demonstrated

* Data Exploration
* Data Preprocessing
* Feature Scaling
* Regression Modeling
* Model Evaluation
* Data Visualization
* Machine Learning Fundamentals

## Conclusion

This project demonstrates an end-to-end machine learning workflow for house price prediction using Linear Regression. It highlights essential concepts such as data preprocessing, model training, performance evaluation, and visualization using Jupyter Notebook.
